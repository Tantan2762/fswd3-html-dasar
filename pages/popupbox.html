Javascript Pop Up Box

Prompt

Prompt sering digunakan jika kita ingin pengguna menginputkan sebuah nilai terlebih dahulu sebelum memasuki sebuah halaman. Ketika prompt box tampil, pengguna harus klik “OK” atau “CANCEL” setelah memasukkan nilai. Jika pengguna klik “OK” maka prompt akan mengembalikan nilai yang dimasukkan. Jika pengguna klik “CANCEL” maka prompt akan mengembalikan nilai null.

Confirm

Confirm digunakan untuk menampilkan pesan konfirmasi kepada pengguna. Confirm box menampilkan pesan dengan dua buah tombol “OK” dan “CANCEL”. Confirm mengembalikan nilai true jika kita tekan “OK” dan false jika kita tekan “CANCEL”.
