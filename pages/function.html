Javascript Function

Basic Function

Merupakan cara yang paling umum digunakan untuk membuat fungsi. Caranya tinggal menambahkan keyword fungsi diikuti dengan nama fungsi yang akan dibuat

Function Expression

Cara lainnya adalah dengan membuat sebuah variabel dan mengisinya dengan fungsi. Fungsi ini merupakan fungsi anonym karena tidak memiliki nama fungsi

Arrow Function

Merupakan salah satu cara penulisan fungsi dalam javascript yang modern dan baru diperkenalkan pada standar ES6. Penulisan dengan menggunakan panah lebih sederhana.
