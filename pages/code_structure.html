Code Structure

Secara umum struktur kode javascript atau yang bisa kita sebut dengan script terditri tiga bagian yaitu statement, semi colon (titik koma) “;” dan komentar (comments). Sebuah statement bisa kita sebut sebagai kalimat tapi dalam kode.

Statement

Statement adalah semua syntax dan perintah-perintah yang menjalankan aksi

Semi Colon

Semi colon atau titik koma merupakan penanda akhir dari statement untuk dieksekusi. Pada javascript titik koma tidak menjadi sebuah kewajiban jika antara statement satu dengan lainnya dipishakan dengan baris baru. Namun jika setiap statement ditulis dalam satu baris maka wajib diakhiri dengan titik koma.

Comment

Semua teks yang terdapat blok comment tidak akan dieksekuis/ diabaikan.Penulisan komentar dalam javascript ada dua, komentar satu baris dan komentar multi baris. Sebuah komnetar satu baris ditandai dengan menambahkan “//” di awal statement, sedangkan komentar multi baris dibungkus dengan tanda “/*…..*/”.
