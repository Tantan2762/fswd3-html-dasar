Javascript Conversion Type

Type Conversion String

Konversi ke tipe data String biasanya diperlukan saat membututuhkan nilai yang berbentuk String. Sebagai contoh jika kita konversi nilai false maka akan berubah menjadi “false”, null menjadi “null” dan sebagianya.

Type Conversion Numeric

Sebenarnya jika kita melakukan operasi matematika sebuah nilai akan otomatis berubah menjadi tipe data numeric. Sebagai contoh jika kita melakukan operasi pembagian kepada bilangan non-numeric maka akan dihasilkan tipe data numeric

Type Conversion Boolean

Konversi Boolean memiliki aturan sebagai berikut:
• Nilai 0, String yang kosong, null, undefined, and NaN, menjadi false.
• Selain itu menjadi true.
